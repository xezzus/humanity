# Humanity PHP Framework

## TEMPLATES

### CSS

```
<?php self::$css->add('name_file'); ?>
```

### JS

```
<?php self::$js->add('name_file'); ?>
```

### APPS

```
<?php self::$apps->nameFolder->nameFile($param1,$param2,$param3); ?>
```

### WIDGET

```
<?php self::$widget->nameFolder->nameFile(); ?>
```
